### Requirements

1. PostgreSQL 13
2. Node 16
3. .env.dev file in root folder

```
NODE_PATH=./
LOG_LEVEL=debug
PORT=8000

DATABASE_URL=postgres://postgres:password@localhost:5432/news_portal
DATABASE_SSL=false

SECRET_COOKIES_KEY=super_secret_key
```

### Quick start

1. yarn
1. yarn dev
